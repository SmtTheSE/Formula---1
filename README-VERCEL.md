# F1 Analytics Dashboard - Deployed on Vercel

This project is a modern F1 analytics dashboard built with Next.js and deployed on Vercel. The site features comprehensive Formula 1 data visualization and analysis.

## Features

- Interactive F1 data visualizations
- Driver statistics and performance analytics
- Historical race data and trends
- Modern UI with responsive design
- Optimized for performance and SEO

## Deployment

This project is designed for easy deployment on Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/your-repo-name)

## Local Development

To run this project locally:

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Project Structure

- `pages/index.js` - Main homepage with F1 analytics
- `styles/globals.css` - Global styling
- `public/` - Static assets like images and scripts
- `package.json` - Dependencies and scripts

## Technologies Used

- Next.js - React framework
- React - UI library
- Vercel - Deployment platform
- CSS - Styling
